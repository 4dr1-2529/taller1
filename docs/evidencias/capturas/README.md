# Evidencias — Capturas del sistema

Almacene capturas de pantalla del sistema web en funcionamiento.

## Qué guardar aquí

| Tipo | Ejemplo de archivo |
|------|-------------------|
| Login exitoso | `login-director-2026-06-04.png` |
| Vista estudiantes CRUD | `director-estudiantes-lista.png` |
| Notas con filtros | `director-notas-bimestre-I.png` |
| Predicción ejecutada | `director-prediccion-resultado.png` |
| Alertas | `director-alertas-lista.png` |
| Mensajería | `mensajeria-comunicado.png` |

## Subcarpetas sugeridas

- `director/` · `profesor/` · `estudiante/` · `login/` · `alertas/`

## Requisitos

- Formato: **PNG** o **JPG**, mínimo 1280×720
- Usar cuentas demo; no exponer DNI real
- Referenciar caso de prueba (ej. TC-DASH-01)
