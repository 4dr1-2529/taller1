# Evidencias — Dashboard

Capturas de KPIs y gráficos por rol.

## Qué guardar aquí

| Rol | Captura |
|-----|---------|
| Director | KPIs totales, gráficos Recharts, distribución riesgo |
| Profesor | KPIs acotados a sus secciones |
| Estudiante | Gauge de riesgo, resumen personal |
