# Datos

Coloque aquí datasets CSV para entrenamiento offline. El script `train.py` genera datos sintéticos si no encuentra un archivo `desercion_dataset.csv`.

Variables alineadas a la tesis:
`promedio_general`, `cursos_desaprobados`, `asistencia_general`, `frecuencia_acceso_lms`, `tiempo_plataforma`, `tareas_ratio`, `participacion_actividades`, `uso_foros`, `disminucion_actividad`, `estado`, `nivel_riesgo`.
