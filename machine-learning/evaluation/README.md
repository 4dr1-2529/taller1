# Evaluación

Métricas de evaluación (accuracy, precision, recall, F1, matriz de confusión) se generan al ejecutar `train.py` y se guardan en `../models/metrics.json` y `../reports/` (si está configurado).

Consulte `GET /metrics` en el servicio ML (puerto 5000) o `GET /api/v1/ml/metrics` en el backend.
